# Azure Agent Utilities

This repository contains miscellaneous utilities related to Azure AI agent
implementations. The scripts here are intended to help test, validate, or
prototype common integration tasks that can support agent workflows.

## Available Scripts

### `scripts/test_graph_calendar.py`

**Description:** Authenticates with Microsoft identity using MSAL device code
flow, then calls Microsoft Graph `GET /me/calendarView` to validate delegated
calendar access.

**Documentation:** [`scripts/test_graph_calendar_README.md`](scripts/test_graph_calendar_README.md)

## Security Notes

Runtime output and local credentials should not be committed. In particular,
`scripts/.token_cache.json` is generated by the Graph calendar test script and
is ignored by this repository.
